Cloud-Native Auto-Scaling Web Application

CI/CD, Monitoring & Infrastructure Automation on AWS

Project Overview

This project demonstrates a cloud-native web application deployed on AWS using Auto Scaling, Load Balancing, CI/CD Pipeline, Monitoring, and Infrastructure as Code.

AWS Services Used

  • Amazon EC2
  • Application Load Balancer (ALB)
  • Auto Scaling Group (ASG)
  • Amazon RDS
  • AWS CodePipeline
  • AWS CodeBuild
  • AWS CodeDeploy
  • Amazon CloudWatch
  • Amazon S3
  • IAM & VPC

Architecture Workflow

User → Route53 → Load Balancer → EC2 Auto Scaling Group → RDS Database

CI/CD Pipeline

GitHub → CodePipeline → CodeBuild → S3 → CodeDeploy → EC2

Deployment Status

Application Successfully Deployed via AWS CI/CD Pipeline
