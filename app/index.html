About Project

This project demonstrates a scalable and highly available cloud-native web application deployed on AWS. The application automatically scales EC2 instances based on CPU utilization using Auto Scaling Groups and distributes traffic through an Application Load Balancer. Continuous Integration and Continuous Deployment (CI/CD) automates application deployment, while CloudWatch provides real-time monitoring and alerts.

AWS Services Used

Amazon EC2

Hosts the web application instances.

Application Load Balancer

Distributes incoming traffic across multiple EC2 instances.

Auto Scaling Group

Automatically adds or removes EC2 instances based on demand.

Amazon RDS

Provides a managed relational database service.

Amazon CloudWatch

Monitors application health, CPU utilization, and system metrics.
