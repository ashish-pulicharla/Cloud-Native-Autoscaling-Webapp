# Project Screenshots

## Overview

This document contains screenshots captured during the implementation and testing of the **Cloud-Native Auto-Scaling Web Application with CI/CD, Monitoring, and Infrastructure Automation on AWS**.

The screenshots provide evidence of infrastructure provisioning, deployment automation, monitoring, security configuration, and successful application deployment.

# Outcome

The screenshots demonstrate that:

- Infrastructure was successfully provisioned using Terraform.
- The application was built using AWS CodeBuild.
- Build artifacts were stored in Amazon S3.
- Deployment was securely performed using AWS Systems Manager (SSM).
- CloudWatch collected metrics and logs.
- CloudWatch Alarms and Amazon SNS provided automated monitoring and notifications.
- The web application was successfully deployed and accessible through the Application Load Balancer.
