# Project Screenshots

## Overview

This folder contains screenshots captured during the implementation, deployment, monitoring, and testing of the **Cloud-Native Auto-Scaling Web Application with CI/CD, Monitoring, and Infrastructure Automation on AWS** project.

The screenshots serve as visual evidence of the successful implementation of the AWS infrastructure, deployment pipeline, monitoring configuration, and application functionality.

## Purpose

These screenshots provide visual proof of:

* Infrastructure provisioning using Terraform
* Deployment automation using AWS CodeBuild and AWS Systems Manager (SSM)
* Application deployment on Amazon EC2
* Auto Scaling and Load Balancer configuration
* Monitoring using Amazon CloudWatch
* Email notifications using Amazon SNS
* Successful deployment and testing of the application
