# Project Documentation

## Overview

This folder contains the complete documentation for the **Cloud-Native Auto-Scaling Web Application with CI/CD, Monitoring, and Infrastructure Automation on AWS** project.

The documentation provides a detailed explanation of the project's architecture, implementation, deployment process, monitoring setup, security measures, testing, and final outcomes.

## Project Highlights

- Infrastructure provisioned using **Terraform**.
- Automated deployment using **AWS CodeBuild** and **AWS Systems Manager (SSM)**.
- Secure architecture with **private EC2 instances** and an **Application Load Balancer**.
- Continuous monitoring using **Amazon CloudWatch**.
- Automated email notifications using **Amazon SNS**.
- Complete technical documentation and implementation details.
